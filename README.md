**Общая информация**

Cервис Mesto: интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

**Технологии**
* flexbox верстка
* grid
* @media использовался для разных разрешений
* Организация файлов выполнена по БЭМ Nested

**Настройки**
* HTML 5
* CSS
* JavaScript
